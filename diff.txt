diffコマンドの検証
----------------------------------------------------
・git pullする前に、リモート追跡ブランチとの差分を確認する
ローカルリポジトリのmasterブランチの最新コミット（HEAD）と、リモート追跡ブランチ（origin/master）を比較する
$ git diff HEAD..origin/main
→【リモート視点でローカルと比較】

・git pushする前に、リモート追跡ブランチとの差分を確認する
上とは逆パターンも可能。ローカルで進めたコミットをpushする前に、リモートリポジトリの状態と比較したいときにはこのように実行します
$ git diff origin/main..HEAD
→【ローカル視点でリモートと比較】
